# Portfolio Tracker Backend

This directory contains the FastAPI backend for the Portfolio Tracker application.

## Documentation

For full project documentation, including setup, API usage, and MCP integration, please refer to the main [Project Documentation](../PROJECT_DOCUMENTATION.md).

## Quick Start

1.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the server**:
    ```bash
    uvicorn main:app --reload --port 8080
    ```

3.  **API Documentation**:
    Open `http://localhost:8080/docs` to view the interactive API documentation.
